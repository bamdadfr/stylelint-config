# stylelint-config-bamdadsabbagh

> [stylelint](https://www.stylelint.io/) config.

## Installation

```shell
yarn add --dev @bamdadsabbagh/stylelint-config
```

## Usage

`.stylelintrc`

```json
{
  "extends": ["@bamdadsabbagh/stylelint-config"]
}
```

It will automatically handle `*.(js|jsx|ts|tsx)` files.
